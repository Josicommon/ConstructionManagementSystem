# INCODM Construction Management System - Brand Guidelines

## Brand Identity
- **Primary Brand Color**: Orange/Amber (#E9922C)
- **Brand Name**: INCODM (construction management focus)
- **Design Philosophy**: Clean, professional, modern construction management interface
- **Target Audience**: Construction professionals, project managers, contractors

## Color Palette

### Primary Colors
- **Brand Orange**: #E9922C (primary actions, logos, highlights)
- **Orange Hover**: #D97706 (hover states)
- **Orange Light**: #FFF7ED (backgrounds, subtle highlights)
- **Orange Dark**: #C2410C (dark accents)

### Text Colors
- **Primary Text**: #1F2937 (headings, important text)
- **Secondary Text**: #6B7280 (body text, descriptions)
- **Muted Text**: #9CA3AF (labels, metadata)

### Neutral Colors
- **Background**: #FFFFFF (main background)
- **Card Background**: #FFFFFF (card backgrounds)
- **Border**: rgba(0, 0, 0, 0.1) (subtle borders)
- **Input Background**: #F3F3F5 (form fields)

## Typography Guidelines

### Hierarchy
- **H1**: 24px, medium weight, brand text dark color
- **H2**: 20px, medium weight, brand text dark color  
- **H3**: 18px, medium weight, brand text dark color
- **H4**: 16px, medium weight, brand text dark color
- **Body**: 16px, normal weight, brand text light color
- **Labels**: 16px, medium weight, brand text dark color

### Font Usage
- Use system fonts for clean, readable interface
- Maintain consistent line height of 1.5 for readability
- Bold text sparingly for emphasis only

## Component Guidelines

### Buttons
- **Primary Button**: Brand orange background (#E9922C), white text, hover to #D97706
- **Secondary Button**: White background, brand orange border and text, hover to light orange background
- **Ghost Button**: Transparent background, brand orange text, hover to light orange background
- **Disabled State**: Reduced opacity (50%), no hover effects

### Cards
- Clean white background with subtle border
- Consistent padding and spacing
- Use orange accents sparingly for status indicators
- Round corners (10px radius)

### Status Indicators
- **In Progress**: Brand orange (#E9922C)
- **Behind Schedule**: Red (#DC2626)  
- **Ahead of Schedule**: Green (#16A34A)
- **Completed**: Gray (#6B7280)
- **Planning**: Yellow (#EAB308)

### Forms
- Light gray background for input fields (#F3F3F5)
- Orange focus ring on active fields
- Clear labels above inputs
- Consistent spacing between form elements

### Navigation
- Clean, minimal sidebar design
- Orange accent for active navigation items
- Clear visual hierarchy
- Professional iconography

## Layout Guidelines

### Spacing
- Use consistent 24px spacing between major sections
- 16px spacing between related elements
- 8px spacing for tight groupings
- Generous whitespace for clean, uncluttered appearance

### Grid System
- Responsive design with mobile-first approach
- Use CSS Grid and Flexbox for layouts
- Maintain proper spacing across all screen sizes
- Maximum content width for readability

### Visual Hierarchy
- Orange color draws attention to primary actions
- Size and weight create clear content hierarchy
- Consistent spacing reinforces organization
- Strategic use of color for status and importance

## Usage Rules

### Do's
- Use brand orange for primary actions and key highlights
- Maintain consistent spacing and typography
- Keep layouts clean and uncluttered
- Use status colors consistently across the application
- Provide clear visual feedback for user interactions

### Don'ts
- Don't overuse the brand orange - use strategically
- Don't use orange for error states (use red instead)
- Don't mix different orange shades randomly
- Don't crowd the interface with too many competing elements
- Don't use low contrast color combinations

## Construction-Specific Elements

### Project Status
- Use color coding consistently for project phases
- Clear progress indicators with brand orange highlights
- Professional iconography for construction elements

### Data Visualization
- Orange color scheme for charts and graphs
- Clear, readable data presentation
- Consistent styling across all visualizations

### Professional Aesthetic
- Maintain construction industry professional standards
- Clean, business-appropriate design
- Focus on functionality and usability
- Modern but not trendy design choices
